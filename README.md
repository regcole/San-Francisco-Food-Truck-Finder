# San-Francisco-Food-Truck-Finder
Outputs a list of current food trucks that are open (at time program is ran) in San Franciso

## Prerequites

- Maven
- JDK 8

## To Run

1. Navigate to root of folder
  ```shell
     cd */SpringFoodTruckFinder
   ```
     
2. Run `Maven` install
  ```shell
   mvn install 
  ``` 
  
 3. Run "mvn springboot:run"
 ```shell 
 mvn spring- boot:run
 ```
