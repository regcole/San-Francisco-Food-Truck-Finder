# Getting Started

## Prerequites

- Maven
- JDK 8

## To Run

1. Navigate to root of folder
    ```shell
        */SpringFoodTruckFinder
    ```
     
2. Run `Maven` install
  ```shell
   mvn install 
  ``` 
  
 3. Run "mvn springboot:run"
 ```shell 
 mvn spring- boot:run
 ```
